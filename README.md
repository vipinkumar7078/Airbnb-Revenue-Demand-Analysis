# 🏡 Airbnb Revenue & Demand Analysis

A complete **Data Analytics Project** built using **Python, SQL, Power BI, and Machine Learning** to analyze Airbnb listing data, identify pricing trends, understand booking demand, and generate actionable business insights.

---

## 📌 Project Overview

This project focuses on analyzing Airbnb listings to answer key business questions such as:

- Which cities generate the highest revenue?
- Which room type is most popular?
- What factors influence Airbnb pricing?
- Which listings have the highest booking demand?
- How can Machine Learning predict listing price and booking demand?

The project combines **Data Cleaning, Exploratory Data Analysis (EDA), SQL Analysis, Interactive Dashboards, and Machine Learning** to provide a complete end-to-end analytics solution.

# 📌 Key Business Insights

- Entire Home/Apartment is the most preferred room type.
- Premium cities have significantly higher average prices.
- Listings with higher reviews generally experience stronger booking demand.
- Professional hosts manage multiple listings efficiently.
- Dynamic pricing can improve occupancy and revenue.
- Machine Learning models support data-driven pricing and demand forecasting.

---

# 💼 Business Recommendations

- Apply dynamic pricing strategies.
- Promote high-performing listings.
- Improve visibility of low-demand properties.
- Optimize pricing in premium markets.
- Use demand prediction to improve occupancy.
